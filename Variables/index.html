Hola Mundo

Let: Permite definir variables limitando su alcance al bloque declaración o expresión donde se está usando
Var: Define la variable global o local sin importar el ámbito del bloque
Global: Fuera de función
Local: Dentro de función
Operadores: Los operadores permiten manipular el valor de las variables, realizar operaciones matemáticas con sus valores y comparar diferentes variables

Operadores de Comparación

  • Mayor: >
  • Menor: <
  • Mayor o igual: >=
  • Menor o igual: <=
  • Asignación: = (asigna valor a la variable)
  • Comparación: == (comprueba si el valor de a es igual al de b, no el tipo de dato)
  • Literal: === (igual tipo de dato y valor)
  • Distinto: !=

Operadores Aritméticos

  • Más: +
  • Menos: -
  • Multipicar: *
  • Dividir: /
  • Módulo: % (te da el resto entero de la división de 2 números)
  • Incremento: ++
  • Decremento: --
  • Exponente: ** (calcula base a la potencia del exponente)

Operadores Lógicos

  • And: &&
  • Or: ||
  • Not: !
