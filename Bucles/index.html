Bucles, loops, iteradores o ciclos

Bucles: Estructura de control que se repite-ejecuta un número indefinido de veces.
  • For: El bucle FOR se utiliza para repetir una o más instrucciones un determinado número de veces. De entre todos los bucles, el FOR se suele utilizar cuando sabemos seguro el número de veces que queremos que se ejecute.
  • While: Mientras la condición sea verdadera, el código del cuerpo del bucle será ejecutado. Se usa para crear bucles infinitos fácilmente, aunque tiene más usos.
  • Do While: Primero se ejecuta el bloque de código y luego se comprueba la condición antes de volverla a ejecutar
    Se usa cuando quieres que el cuerpo del bucle se ejecute, al menos, una vez sin importar si la condición es verdadera.

Iteración: Cada ejecución del cuerpo del bucle.

Romper el bucle:

Se puede forzar una salida del bucle en cualquier momento usando la directiva especial break.
La directiva continue es una “versión más ligera” de break. No detiene el bucle completo. En su lugar, detiene la iteración actual y fuerza al bucle a comenzar una nueva (si la condición lo permite). Podemos usarlo si hemos terminado con la iteración actual y nos gustaría movernos a la siguiente.
