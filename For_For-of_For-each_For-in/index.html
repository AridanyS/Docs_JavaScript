• For: (Creación de un bucle).
  • ForEach: (Solo Arrays). ForEach es un método que itera solamente sobre los elementos que contiene un array y ejecuta una función determinada en cada uno de ellos. arr.forEach(function callback(currentValue, index, array) { // tu iterador }[, thisArg]); Documentación: https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach
  • For in: (Solo objetos) (No se debe usar para iterar en un array). Itera sobre propiedades enumerables de un Objeto. for (variable in objeto) instrucción
  • For of: Itera sobre elementos que no sean necesariamente arrays o cualquier objeto iterable (String, Array, ...) y sobre las propiedades del mismo. Incluso elementos que aún no existen. for (variable of iterable) { statement }
